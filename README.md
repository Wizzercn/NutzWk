NutzWk 2.0 纯净版
======

基于Nutz的开源企业级开发框架


NutzWk 2.0 新特性：
*   集成Shrio权限框架
*   集成Ehcache缓存
*   支持语言国际化
*   支持注解式事务
*   支持动作链
*   支持注解式日志系统[Coding..]
*   支持插件式加载[Coding..]
*   集成Email服务
*   集成Quartz定时任务
*   集成Lucene搜索引擎[Coding..]
*   采用Bootsrtap皮肤(Angle)
