<li>  Nutz v1.r.57</li>

<p>NutzWk 3.x 新特性：</p>

<ul>
<li>  集成Shiro权限框架</li>
<li>  集成Ehcache缓存</li>
<li>  集成Redis</li>
<li>  支持语言国际化</li>
<li>  支持注解式事务</li>
<li>  支持动作链</li>
<li>  支持注解式日志</li>
<li>  支持插件式加载</li>

<li>  集成Beetl模板引擎</li>
<li>  后台管理界面采用Pjax+Bootstrap</li>
<li>  自定义路由</li>
</ul>

<ul>
